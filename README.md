## :computer: Projeto
Aplicação web para catalogar uma lista de filmes com operações de inserção, pesquisa, atualização e exclusão de registros utilizando Angular CLI. Projeto teve seu código fonte com base no curso [Técnicas avançadas em Angular 8](https://digitalinnovation.one/cursos/tecnicas-avancadas-em-angular-8).  

## :wrench: Recursos Utilizados
- [Visual Studio Code v1.47.0](https://code.visualstudio.com/).
- [Angular CLI v10.0.0](https://cli.angular.io/).
- [Angular Material v10.0.1](https://material.angular.io/).
